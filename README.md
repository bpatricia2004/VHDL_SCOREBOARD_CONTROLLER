# FPGA Scoreboard Controller
VHDL FPGA scoreboard controller with button debouncing and 7-segment display driver.
VHDL implementation of a scoreboard controller running on FPGA.

## Features
- Button debouncing
- Score increment / decrement
- 7-segment display driver
- Reset functionality
- 3-second wait state

## Architecture
Buttons → Debouncer → Control Logic → Score Counter → 7-Segment Display
